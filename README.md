### Features
  - [x] Sign up user
  - [x] Log in user
  - [x] See a list of all the todos for user
  - [x] Create new todos for user
  - [x] Edit a todo description
  - [x] Delete a todo
  - [x] Mark todos as complete or incomplete for user
  - [x] Reorder their todos via drag and drop.
